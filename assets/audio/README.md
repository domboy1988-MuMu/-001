# 背景音樂

本資料夾已含兩個檔案：

    bgm.mp3           ← 實際使用（無縫循環版，2:20）
    bgm-original.mp3  ← 你的原始檔（2:27，備份用）

## 為什麼要另做循環版

原始檔在 143 秒後有淡出，直接循環會出現：
音樂漸弱到幾乎無聲（-39.6 dB）→ 突然跳回全音量（-18.3 dB），
接點落差達 21.3 dB，聽起來像斷掉再重播。

循環版的處理：
1. 剪掉結尾 4 秒的淡出
2. 將尾端 3 秒與開頭 3 秒以等功率曲線交叉淡接

處理後接點落差降至 5.3 dB，循環時聽不出斷點。

若你想改用原始檔，把 bgm-original.mp3 改名為 bgm.mp3 覆蓋即可。

## 運作方式

- 無限循環播放，整套流程不中斷（單頁應用，換頁不會重播）
- 於「輕觸水晶球」時啟動——瀏覽器禁止未經互動的自動播放，
  而這一下本來就要開啟旁白音效，因此不需額外要求使用者操作
- 音樂音量恆定，不隨影片起伏（先前版本會忽大忽小，已修正）
- 影片音軌為環境音效，以較低音量與音樂分層，不互相爭搶
- 左上角有靜音鈕；未放置檔案時該鈕會自動隱藏

## 建議規格

- 格式：MP3（相容性最好）。也可另放 bgm.ogg 作為備援
- 長度：3 分鐘以上較佳，循環接點才不明顯
- 音量：不需預先調小，程式會控制
- 檔案大小：建議 3MB 以內，避免拖慢載入

## 想調整音量

編輯 src/app.js 最上方：

    const BGM_VOLUME = 0.30;   // 背景音樂（主導）
    const VID_VOLUME = 0.42;   // 影片音效（襯底）

影片音效的數值雖然較高，但影片素材本身遠比音樂輕
（-23～-38 dB，音樂為 -16.7 dB），混音後仍是音樂主導約 4～18 dB。
若覺得音效太吵，把 VID_VOLUME 調到 0.25 左右；
想完全只留音樂，設為 0 即可。
