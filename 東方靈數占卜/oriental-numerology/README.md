# 東方靈數占卜

## 兩個版本

- demo.html    免費版（15 頁，關鍵處收束導向銷售頁）
- unlock.html  付費版（21 頁，全部揭曉＋數字檢測工具）

兩者共用 src/app.js 與 src/app.css，只差入口檔的 window.UNLOCKED 旗標。
日後修改只需改 app.js，兩版自動同步。

⚠️ 入口檔請使用 window.UNLOCKED = true/false，
   不要改成 const 或 let——會與 app.js 的宣告衝突而導致整頁空白。

## 背景音樂

放置 assets/audio/bgm.mp3 即可自動循環播放。
於「輕觸水晶球」時啟動（瀏覽器禁止未經互動的自動播放），
有旁白的影片播放時會自動降低音量，左上角有靜音鈕。
未放置檔案時靜音鈕會自動隱藏，不影響其他功能。

## 試玩方式（一定要用本機伺服器）

直接雙擊 demo.html 會失敗——瀏覽器的 file:// 協定不允許網頁載入本機影片。
必須這樣開：

1. 在本資料夾的檔案總管路徑列打 cmd 按 Enter
2. 輸入：py -m http.server 8000
3. 瀏覽器開啟：http://localhost:8000/demo.html

黑視窗要保持開著。要關閉伺服器按 Ctrl+C。
（找不到 py 就改用 python，或雙擊 start-server.bat）

## 資料夾可以隨便搬

所有路徑都是相對的，整個資料夾放哪、叫什麼名字都能跑，
只要 demo.html、src/、assets/ 三者維持在同一層即可。

## 調整水晶球熱區位置

1. 開啟頁面後按鍵盤 C，熱區會顯示成金色虛線圓
2. 看它有沒有蓋住影片裡的水晶球
3. 沒對準就編輯 demo.html 最上方的 HOTSPOTS：
       x = 圓心橫向位置（佔畫面寬度 %）
       y = 圓心縱向位置（佔畫面高度 %）
       size = 圓的直徑（佔畫面寬度 %）
4. 重新整理，再按 C 檢查

## 結果頁的「日／月」返回熱區

預設關閉，因為 v4 影片右上角未必真的有太陽。
若你的影片有，把 demo.html 裡的 SUN_HOTSPOT_ENABLED 改成 true 即可。
關閉時，結果頁下方會有「重新測算」文字連結。

## 其他可調參數（都在 demo.html 最上方）

CALC_MS   計算動畫長度，預設 9000 毫秒
PROGRESS  計算過程的進度文字

## 目前流程

輕觸水晶球 → 輸入生日 → 送出 → 計算 → 數字浮現於水晶球內
  → 魔法書（11 頁，可左右翻）→ 查看完整報告 → 銷售頁

## 魔法書翻頁方式

- 手機：左右滑動
- 電腦：鍵盤方向鍵，或點下方箭頭
- 也可直接點下方圓點跳頁

頁面順序：命格總覽 / 個人特質 / 內在與外在 / 天生的數字 /
天生缺乏什麼 / 人生課題 / 合適的顏色 / 契合的人 /
工作與賺錢 / 流年 / 收尾

## 兩套流派並存

本專案同時使用兩套數字系統，互相印證而非取代：

1. 生命靈數（西洋數字學）— 全部數字相加歸一，看人格原型
   檔案：src/engine.js、src/content.js
2. 數字易經・八星磁場（源自易經八卦）— 兩兩相鄰拆解，看能量分布
   檔案：src/magnetic.js、src/magnetic-content.js

八星磁場的價值在於可套用到後天數字（手機、車牌、住址），
因此付費後的「數字檢測工具」是回購與口碑的主要來源。

註：八星磁場的拆解方式各流派略有差異，本專案採
「剔除 0 與 5 後，相鄰兩位數滑動配對」，理由見 magnetic.js 註解。
